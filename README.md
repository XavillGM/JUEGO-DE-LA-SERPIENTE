JUEGO DE LA SERPIENTE
ESTUDIANTE: KEVIN XAVIER MERO MANTUANO
Explicación del sistema
El videojuego inicia mostrando una ventana donde aparece la serpiente.
El jugador utiliza las flechas del teclado para moverla.
Cada vez que la serpiente consume un alimento:
•	aumenta de tamaño;
•	incrementa el puntaje;
•	se genera una nueva comida en una posición aleatoria.
El juego termina cuando la serpiente choca contra los bordes de la pantalla o contra su propio cuerpo.
Durante el desarrollo se utilizaron:
1. Importación de librerías
   - pygame
   - random
   - sys
2. Inicialización del programa
   - pygame.init()
   - creación de ventana
   - configuración de colores, tamaño y fuente
3. Funciones del sistema
   - mostrar_puntaje()
   - dibujar_serpiente()
   - crear_comida()
   - menu()
   - juego()
   - game_over()
4. Ejecución principal
   - menu()
